# RecallOS Privacy and Routing Policy

## Principles
- local-first by default
- no silent network usage
- explicit user control

## Execution modes

### Local only
No network requests allowed.

### Local preferred
Use local execution unless explicitly required.

### Balanced
Allow automatic routing.

### Best quality
Allow network for higher quality results.

## Rules
- local-only mode must be strictly enforced
- routing decisions must be visible to user
- no hidden data transmission

## Data handling
- raw content stored locally
- no automatic cloud sync
- optional sync must be explicit
